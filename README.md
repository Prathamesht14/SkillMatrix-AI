# 🚀 SkillMatrix AI

SkillMatrix AI helps students and professionals bridge the gap between their current skills and industry demands through smart analysis, real-time feedback, and guided career roadmaps.

---

## 📌 Overview

SkillMatrix AI is a full-stack web application that analyzes user skills, compares them with industry requirements, and provides actionable insights to improve career readiness.

---

## ✨ Features

- 🔍 Skill Gap Analysis
- 📊 Role-based Skill Matching
- 🧠 Smart Recommendations
- 🌐 Live Web Application
- ⚡ Fast and Interactive UI

---

## 🏗️ Tech Stack

### Frontend
- HTML, CSS, JavaScript

### Backend
- Node.js (Express)
- Hosted on **AWS EC2**
- Managed using **PM2**

### Cloud & DevOps
- AWS EC2 (Compute)
- Elastic IP (Static IP)
- Git & GitHub (Version Control)

---

## ⚙️ How It Works

1. User enters skills and target role
2. Frontend sends request to backend API
3. Backend processes data and compares with dataset
4. Returns:
   - Skill Match %
   - Missing Skills
   - Recommendations
